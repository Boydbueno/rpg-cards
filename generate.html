RPG card generator

Experimental user interface! This user interface is experimental and largely non-functional. The only thing that works right now are the "load sample cards", "load cards from file", "save cards", and "generate cards" buttons.
Warning! Some RPG systems are protected by copyright. Even if some spell/item data is freely available, it does not mean you may redistribute cards containing such data.
If you need help on how to use this tool, go visit http://crobi.github.io/rpg-cards/.
